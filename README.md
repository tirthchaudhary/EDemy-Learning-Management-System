# EDemy — Full-Stack e-Learning Platform (MERN)

EDemy is a modern, high-performance e-Learning web application built using the MERN stack (MongoDB, Express, React, Node.js). It supports separate flows for Students and Educators with a polished, fully-responsive dashboard interface.

### Key Features
* 🔐 **Secure Authentication:** Complete registration, login, and profile update functionality using JWT & HTTP-only cookies.
* 📧 **Password Recovery:** Secure forgot/reset password system utilizing Node.js `crypto` tokens, Mongoose timers, and Nodemailer with SMTP.
* 👨‍🏫 **Educator Dashboard:** Analytics panels, course content builders (chapters/lectures), enrollment list managers, and student query tools.
* 📚 **Student View:** Interactive video course players, review and rating modules, and lecture-by-lecture progress tracking.
* 💳 **Payment Gateway:** Razorpay transaction processing for smooth student enrollments.
* 💬 **Doubt Solver:** Dynamic student doubt query threads with real-time educator replies.
